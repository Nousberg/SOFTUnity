ГЛАВА 1. ФИЗИКА (DEFORMATION)



Enable Deformation: разрешает деформацию. Off — почти жёсткое тело.
Enable Plasticity: включает постоянную деформацию по площади (faces).
Area Compliance: податливость площади треугольников. 0 — жёстко, больше — мягче.



ГЛАВА 2. ФИЗИКА РЁБЕР (BEAM DEFAULTS)



Compliance: податливость рёбер по умолчанию (меньше = жёстче).
Damping: демпфирование рёбер по умолчанию (гасит колебания).



ГЛАВА 3. ФИЗИКА РЁБЕР (BEAMS SELECTED)



Compliance: податливость выбранных рёбер.
Damping: демпфирование выбранных рёбер.



ГЛАВА 4. ФИЗИКА УЗЛОВ (NODES)



Mass: масса выбранных узлов.
Is Pinned: закрепляет узлы (не двигаются).



ГЛАВА 5. ДАВЛЕНИЕ



Internal Pressure: внутреннее давление для «надувания».



ГЛАВА 6. СТОЛКНОВЕНИЯ



Enable Collision: включает столкновения.
Collision Radius: радиус коллайдеров узлов.
Friction: трение при контакте.
Collision Layers: слои, с которыми есть столкновения.
Enable Self-Collision: самостолкновения узлов внутри тела.



ГЛАВА 7. MATTER



Restitution: упругость, отскок
Collision Damping: потеря энергии при ударе
